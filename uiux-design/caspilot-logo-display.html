Caspilot — the heading dial

An instrument bezel shaped like a C, with a single slim compass needle pivoting from center and pointing out through the gap. The needle and its tip tick are the only amber. A pilot setting a heading — one accountable, authorized point.

USAGE

Clear-space & minimum size

Specification

clear spaceKeep empty margin equal to the ring’s stroke-radius (≈ glyph ⅓) on all four sides. Nothing intrudes inside that band.
minimum sizeGlyph 16px (use the favicon variant below 24px). Wordmark lockup minimum 96px wide.
accentAmber #FF5A1F appears only on the needle and tip tick — never the ring or wordmark.
ring colorcurrentColor — inherits text color (#ECECEE on dark, #0A0A0B on light).
recolorSet CSS --accent once to retint the needle; set color for the ring.
neverNo gradients inside the mark, no second accent, no rotation of the needle off the gap axis.
